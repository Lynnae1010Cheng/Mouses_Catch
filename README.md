 畢竟忙著抓老鼠，可沒有時間看老鼠能換多少分。

# 執行方式
1. 將素材放在與程式碼同樣的資料夾，執行程式碼就會加入角色物件，並直接開始遊戲。

2. 以pygame套件完成2D小遊戲，使用 while就能讓遊戲不斷刷新。

3. 以滑鼠點擊，便能得分，在時間倒數結束後，清空角色。
